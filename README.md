# Bhutan Gift Transparency System (BGTS)

A web application for gift declaration and management, compliant with Bhutan's Gift Rules 2017.

## Features
- Gift declaration with 24-hour deadline tracking
- Prohibited source checking
- Penalty calculation (Rule 37)
- Gift review workflow for administrators
- Mobile-responsive design
- Full compliance with Gift Rules 2017
